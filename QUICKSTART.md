# Быстрый старт - Лабораторная работа №3

## Быстрая демонстрация

```bash
# 1. Просмотр справки
make help

# 2. Сборка проекта
make

# 3. Запуск программы
make run

# 4. Запуск тестов
make test
```

## Демонстрация инкрементальной сборки

```bash
# Полная сборка
make clean && make

# Проверка времени модификации
stat -c "%y %n" bin/obj/*.o bin/matrix_rotate

# Изменение одного файла
touch src/main.c

# Пересборка (только main.o будет перекомпилирован!)
make

# Проверка времени снова
stat -c "%y %n" bin/obj/*.o bin/matrix_rotate
```

**Результат:** только `main.o` изменился!

## Динамическая компиляция

```bash
# AT&T синтаксис
make SYNTAX=att

# С отладкой
make DEBUG=yes

# С оптимизацией
make COMPILE_FLAGS="-O2"

# Комбинация
make SYNTAX=att DEBUG=yes COMPILE_FLAGS="-O2"
```

## Полезные команды

```bash
make info       # Конфигурация сборки
make clean      # Очистка объектных файлов
make distclean  # Полная очистка
make rebuild    # Пересборка с нуля
```
